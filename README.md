# Мои Bash Скрипты (My Bash Scripts)

Набор полезных Bash-скриптов для автоматизации настройки системы, конфигурации серверов и повседневных задач DevOps.

## 🚀 Быстрый запуск

Вы можете запустить скрипт настройки SSH напрямую из терминала одной командой:

```bash
bash <(curl -Ls https://raw.githubusercontent.com/clarkxkent/my-bash-scripts/refs/heads/main/config_ssh.sh)
```

## 🛠️ Описание скриптов

### 1. Настройка SSH (`config_ssh.sh`)
Скрипт автоматизирует базовую конфигурацию и защиту SSH-сервера.
* **Что он делает:**
  * Изменяет стандартный порт SSH.
  * Отключает вход по паролю для root-пользователя.
  * Настраивает аутентификацию по SSH-ключам.
  * Перезапускает службу SSH для применения изменений.

## 📋 Системные требования

* **Операционная система:** Ubuntu / Debian / CentOS (Linux).
* **Утилиты:** `curl`, `bash`, `sudo` права.

## ⚙️ Локальная установка и запуск

Если вы хотите склонировать весь репозиторий и изучить код локально:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com
   ```
2. Перейдите в директорию:
   ```bash
   cd my-bash-scripts
   ```
3. Сделайте скрипт исполняемым:
   ```bash
   chmod +x config_ssh.sh
   ```
4. Запустите его:
   ```bash
   ./config_ssh.sh
   ```
